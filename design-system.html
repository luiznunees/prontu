Tokens CSS

Use estes tokens no seu CSS:

/* Cores */
--color-ink
--color-paper
--color-accent
--color-accent-soft
--color-blue
--color-yellow
--color-green
--color-gray-100
--color-gray-300
--color-gray-500
--color-gray-700

/* Tipografia */
--font-display
--font-body
--font-mono

/* Espaçamento */
--space-xs --space-sm --space-md --space-lg --space-xl --space-2xl --space-3xl

/* Border Radius */
--radius-sm --radius-md --radius-lg --radius-xl --radius-full

/* Sombras */
--shadow-sm --shadow-md --shadow-lg --shadow-xl

/* Transições */
--transition-fast --transition-normal --transition-slow

Como Usar

Adicione as variáveis CSS no seu projeto:

<link href="https://fonts.googleapis.com/css2?family=DM+Mono:wght@400;500&family=DM+Sans:wght@400;500;600;700&family=Syne:wght@400;500;600;700;800&display=swap" rel="stylesheet">

<style>
  :root {
    --color-ink: #0D0D0D;
    --color-paper: #F7F5F0;
    --color-accent: #FF4D00;
    /* ... demais tokens */
  }
</style>
