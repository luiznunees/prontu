🗺️ Fluxo de Usuário - Prontu

Mapeamento do que um visitante pode fazer no sistema

👤 Visitante Não Logado

Primeiro Acesso

Usuário queaccessa o site pela primeira vez, sem conta criada.

Grátis 1 trabalho/mês
1
Acessa Landing Page
Visualiza página inicial com:
  • Hero com copy de marketing (valores, diferenciais)
  • Seção "Como Funciona" (3 passos)
  • Comparativo ChatGPT vs Prontu
  • Planos de preços (Starter R$14,90 / Popular R$34,90 / Pro R$59,90)
  • Botão "Entrar" no header
↓
2
Preenche Formulário de Trabalho
No formulário, preenche:
  • Enunciado ou tema do trabalho (obrigatório, mín 20 chars)
  • Nome completo do aluno (obrigatório)
  • Escola (obrigatório)
  • Disciplina (dropdown com 13 opções)
  • Série (ex: "2 Médio")
  • Turma (opcional)
  • Cidade (opcional)
  • Observação personalizada (opcional)
  • Estilo do documento (escolha visual da capa)
↓
3
Clica em "Gerar meu trabalho"
Sistema detecta que não está logado
→ Abre Modal de Autenticação
if (!user) { setIsAuthModalOpen(true); }
↓
4
Modal de Login/Cadastro
Pode criar conta ou fazer login via:
  • Email + Senha
  • Google OAuth
⚠️ Ao criar conta, ganha 1 trabalho grátis por mês

✅ Usuário Logado (Plano Grátis)

1
Preenche formulário e gera trabalho
Após login, pode gerar trabalho diretamente
✓ Primeiro trabalho do mês: gratuito
↓
2
Tela de Resultado
Após geração (~40s), vê:
  • Confetes de celebração
  • Botão principal: "⬇ Baixar PDF"
  • Ferramentas gratuitas:
    • Analisar Originalidade - Detecta se parece IA
    • Humanizar Texto - Reescrever para parecer humano
  • Rodapé: créditos restantes + botão "Gerar Outro"
↓
3
Tenta gerar segundo trabalho no mesmo mês
Sistema verifica limite
if (plano === "gratis" && trabalhos_gerados_mes >= 1) { setIsPayModalOpen(true); }
→ Abre Modal de Pagamento
↓
4
Modal de Pagamento
Oferece planos:
Plano Preço Trabalhos
Starter R$ 14,90 5 trabalhos
Popular ⭐ R$ 34,90 15 trabalhos
Pro R$ 59,90 30 trabalhos
Pagamento via PIX via AbacatePay

💎 Usuário com Créditos (Pagante)

1
Dashboard
Acessível via header "🏠 Meu Dashboard"
  • Mostra nome do usuário
  • Quantidade de créditos restantes
  • Histórico de trabalhos gerados
  • Botão para comprar mais créditos
↓
2
Gera trabalhos livremente
Cada trabalho subtrai 1 crédito
✓ Pode usar todas as ferramentas:
  • Analisar originalidade
  • Humanizar texto (3 níveis: leve, médio, forte)
  • Baixar PDF original ou humanizado
↓
3
Créditos acabaram
Ao tentar gerar novo trabalho com 0 créditos
→ Abre Modal de Pagamento novamente

🎯 Estados do Usuário no Sistema

Estado Condição Ação Restrita
Visitante Não logado Não pode gerar trabalho (pede login)
Grátis Logado, 1º trabalho/mês Pode gerar 1x, depois pedir pagamento
Pagante Comprou créditos Pode gerar até créditos terminarem
Sem créditos Créditos = 0 Bloqueado até comprar mais
✅ Fluxo mapeado! Este é o comportamento atual baseado nos arquivos do projeto.
